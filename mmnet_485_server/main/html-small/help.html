URLs

  • /item.cgi?name=adc0 - read item value
  • /item.cgi?name=dig0&value=1 - write item value

Item names

  • adc - Analogue inputs
  • dig - Digital I/O, bits
  • temp - Temperature sensors (1-wire), ordered (see Config page)
  • utemp - Temperature sensors (1-wire), unordered
  • pwm - Analogue outputs
  • dht-t - DHT11/22 temperature
  • dht-h - DHT11/22 humidity

Pins

  • PB0 - #ss for onboard serial Di/Do extension
  • PB1-3 - /sck/mosi/miso
  • PB5 - #ss internal flash
  • PB6-7 - Tx Enable usart 0,1 (half duplex 485)
  • PD0-1 - SCL, SDA (2wire/i2c, bmp180)/Di/Do
  • PD2-3 - RxD1,TxD1/Di/Do
  • PD4-7 - Di/Do or PD2-7 - 1Wire multiple bus
  • PE0-1 - RxD0, TxD0/Do/Di
  • PE2-3 - Do/Di
  • PE4 - Fail LED
  • PE5 - Ethernet interrupt
  • PE6-7 - Di/Do
  • PF0-7 - Ai/Di/Do
  • PG3 - I/O LED
  • PG4 - 1Wire (2401, single bus 18B20, etc)

Defaults

  • IP address - 192.168.88.127
  • MAC - 02:06:98:00:00:00
  • Serial speed - 38400
  • Debug serial - UART1 (PD2-3)
