URLs:
  • /item.cgi?name=adc0 - read item value

    Item names:
  • adc - Analogue inputs
  • dig - Digital I/O, bits
  • temp - Temperature sensors (1-wire), ordered (see Config page)
  • utemp - Temperature sensors (1-wire), unordered
  • pwm - Analogue outputs
  • dht-t - DHT11/22 temperature
  • dht-h - DHT11/22 humidity
